# 📱 QR Code Generator

A simple and responsive QR Code Generator built using **HTML**, **CSS**, and **JavaScript**. This application allows users to generate QR codes instantly by entering any text, URL.



## 🌐 Live Demo

https://bespoke-stardust-89f41d.netlify.app/




## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- QR Code API 

## 📂 Project Structure

```text
QR-Code-Generator/
│── index.html
│── style.css
│── main.js
│── README.md

```


## 📖 How to Use

1. Enter any text or URL.
2. Click the **Generate QR Code** button.
3. Your QR code will be generated instantly.
4. Scan it using any QR scanner.
